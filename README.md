# Personal Portfolio Website

A modern and fully responsive personal portfolio website built using HTML5, CSS3, and JavaScript to showcase projects, technical skills, certifications, and contact information.

---

## Live Demo

Frontend: https://sakshisaog.github.io/

---

## Features

- Fully Responsive Layout
- Animated UI Interactions
- Custom Cursor Effects
- Smooth Scroll Navigation
- Scroll Reveal Animations
- Interactive Contact Section
- Modern UI Design
- Mobile-Friendly Interface

---

- Flexbox
